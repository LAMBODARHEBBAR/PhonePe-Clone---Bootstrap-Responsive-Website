# PhonePe Clone - Bootstrap Responsive Website

## 📌 Project Overview

This project is a responsive PhonePe landing page clone created using HTML, CSS, and Bootstrap 5.

The website replicates the modern UI design of the PhonePe homepage and demonstrates responsive web design principles.

### The project includes:

* Responsive navigation bar
* Hero section
* Features section
* Testimonials/Press section
* Mobile-friendly layout using Bootstrap Grid System

---

## 🚀 Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Google Fonts
* Ionicons

---

## 📱 Responsive Design

The project is designed to work properly on:

* Desktop
* Tablet
* Mobile devices

Media queries are used to adjust:

* Image positioning
* Text alignment
* Font sizes
* Layout structure

---

## ✨ Features

* Fully responsive design
* Bootstrap Grid System
* Responsive navigation bar
* Hero section with call-to-action buttons
* Feature cards with icons
* Press/Testimonial section
* Mobile and tablet compatibility
* Clean UI inspired by PhonePe
