# PhonePe-Clone---Bootstrap-Responsive-Website
📌 Project Overview

This project is a responsive PhonePe landing page clone created using HTML, CSS, and Bootstrap 5.
The website replicates the modern UI design of the PhonePe homepage and demonstrates responsive web design principles.

🚀 Technologies Used
HTML5
CSS3
Bootstrap 5
Google Fonts
Ionicons

📱 Responsive Design

The project is designed to work properly on:

Desktop
Tablet
Mobile devices

▶️ How to Run the Project
Download or clone the repository
Open the project folder
Run index.html in your browser
